newSessionMiddleware

  • newSessionMiddleware(sessionLevel: SessionLevel, sessionInfoSource: SessionInfoSource, env: Env, identityClient: IdentityClient): express.RequestHandler
    • Defined in server/session-middleware.ts:52

    Create a connect middleware which populates the incoming request's RequestWithIdentity.session property with a Session object. It does this by looking at either cookies or headers in the HTTP request, extracting authentication information and using this to retrieve data about the session and potentially user from the identity service. It also re-attaches the auth info to the response, as either a cookie or a header.

    Parameters

    • sessionLevel: SessionLevel

      how much of a session to expect to exist.

    • sessionInfoSource: SessionInfoSource

      where to extract the session from the request, and where to place the session info on the response.

    • env: Env

      the environment the code is running in.

    • identityClient: IdentityClient

      an IdentityClient for communicating with the identity service.

    Returns express.RequestHandler

    A connect middleware of type {@link express.RequestHandler}.

setSessionTokenOnResponse

  • setSessionTokenOnResponse(res: express.Response, rightNow: Date, sessionToken: SessionToken, sessionInfoSource: SessionInfoSource, env: Env): void
    • Defined in server/session-middleware.ts:222

    Attach the given SessionToken to the request. Depending on {@link sessionInfoSource}, this means either setting a cookie to the serialized content of the token, or setting a header to the same value. The cookie is http only, secure in production, does not expire and has the sameSite attribute to protect against XSS attacks.

    Parameters

    • res: express.Response

      the response to attach the info to.

    • rightNow: Date

      the request's time

    • sessionToken: SessionToken

      the token to attach to the request.

    • sessionInfoSource: SessionInfoSource

      where to place the session info on.

    • env: Env

      the environment this code is running in.

    Returns void
